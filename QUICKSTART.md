# Quick Start Guide 🚀

Get Pradofy Study Tracker running in 3 minutes!

## Installation (Windows/macOS/Linux)

### Step 1: Install Python Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Run the Application
```bash
python main.py
```

That's it! The application will launch.

## First Time Use

### Add Your First Session
1. Go to the **Sessions** tab
2. Fill in the form:
   - **Subject**: What you're studying
   - **Start Time**: When you started (e.g., `02:30 PM`)
   - **End Time**: When you finished (e.g., `04:00 PM`)
   - **Date**: Today's date appears by default
3. Click **Add Session**
4. Confirm in the popup

### View Your Progress
1. Go to the **Dashboard** tab
2. See your stats at the top
3. Explore the three charts below

## Common Tasks

### Export Your Data
1. Go to **Sessions** tab
2. Click **Export Sessions**
3. A CSV file is saved automatically

### Edit a Session
1. Click on any session in the table
2. Fields auto-fill
3. Change what you need
4. Click **Edit Session**

### Delete a Session
1. Click on the session to select
2. Click **Delete Session**
3. Confirm deletion

## Tips

- **Time Format**: Always use `hh:mm AM/PM` format
  - ✅ Correct: `2:30 PM`, `09:15 AM`
  - ❌ Wrong: `14:30`, `2:30pm`

- **Subject Names**: Use consistent names for better analytics
  - Good: "Math", "Python Programming", "Biology"
  - Avoid: "studying", "work", "stuff"

- **Regular Data**: The more consistent you track, the better your analytics
